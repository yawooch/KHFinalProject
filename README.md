## 개발목표
-	공공데이터를 활용하여 유의미한 사이트를 구현한다
-	부트스트랩을 사용하여 반응형 사이드를 구현한다.
-	GiIt충돌 가능 부분을 예상하고 협의하여 Git충돌을 예방한다.
-	화면부터 Controller, DB 까지 전반적으로 구현한다.
